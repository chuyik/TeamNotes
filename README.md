TeamNotes
=========

It's used for sharing knowledge with team members.

You can post notes using markdown, and add some comments as well.

##Version##

Ruby version: 1.9+

Rails version: 4.1

##Installation##
  ```
  bundle install
  bundle exec rake db:create
  bundle exec rake db:migrate
  ```
